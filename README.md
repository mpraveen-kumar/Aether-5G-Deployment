# Aether-5G-Deployment
This repository documents my research on building and analyzing a private 5G infrastructure using the Aether On-Ramp framework. It includes deployment steps, configurations, and performance evaluation of a 3GPP-compliant 5G core network.

🎯 Project Objectives
-Deploy a functional 3GPP-compliant 5G Core (SD-Core) on bare-metal infrastructure.
-Analyze key network performance metrics such as latency, throughput, and resource utilization.
-Investigate the potential integration of AI-driven anomaly detection for telecom KPIs.

🛠️ Infrastructure & Tools
-Platform: CloudLab (Utah / Clemson clusters)
-Hardware Profile: single-pc-ubuntu (Bare Metal)
-Operating System: Ubuntu 22.04 LTS
-Technologies & Tools: Docker, Kubernetes (RKE2), Ansible, Aether On-Ramp
